## Description (copy below)

### Rentu — The Modern Rental Marketplace for Chile

A production-ready rental property marketplace that connects landlords directly with tenants in Chile. No brokers, no commissions for individuals. Built with a modern tech stack, fully audited, and monetization implemented — ready to generate revenue from day one.

### The Opportunity

- Chile has **2M+ rental properties** with no dominant digital platform
- Portalinmobiliario charges $30-100 USD per listing — Rentu is **free for individuals**
- Growing demand for direct landlord-tenant connections
- Rental market in Chile growing 8% annually

### What You're Buying

**Complete web platform:**
- 23 pages, 26 components, 10 API endpoints
- 43 commits of production-quality code
- Fully audited (security, performance, accessibility, SEO)
- Zero bugs, zero console errors
- Chilean legal compliance (Ley 19.628, Ley 19.496, Ley 21.719)

**Tech stack:**
- React 18 + Vite 5 + Tailwind CSS 3
- Supabase (auth, database, storage, realtime)
- Flow.cl (Chilean payment gateway — already integrated)
- Resend (transactional emails)
- Vercel (hosting — free tier)
- Leaflet (maps — free)
- PWA-ready (installable from browser)

**3 revenue streams already built:**
1. Featured listings: $2,990 - $19,990 CLP per property (7/30/90 days)
2. Broker plans: $14,990 - $59,990 CLP (per listing / monthly / unlimited)
3. Infrastructure ready for: rent collection, tenant screening, insurance partnerships

### Key Features

**For tenants:**
- Advanced search with 15+ filters
- 3 view modes: list, map, gallery
- Property comparison (side-by-side)
- Neighborhood data (population, avg rent, safety, transit)
- Connectivity score per property
- Direct WhatsApp contact with landlord
- Rent affordability calculator
- Save searches + favorites

**For landlords:**
- Free publishing (unlimited for individuals)
- Photo + video upload
- Analytics dashboard (views, contacts, favorites)
- Featured listing boost (paid)

**Platform:**

- SEO landing pages for 15 Chilean cities
- Admin panel with stats, user management, reports
- Onboarding flow for new users
- Newsletter signup
- Contact form
